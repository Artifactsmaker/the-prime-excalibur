## English

The Prime Excalibur is a local Operator Intelligence console. It applies bundled O.i operator specifications before sending a prompt to the selected AI assistant.

### Run

You can use the direct web version:

```text
https://artifactsmaker.github.io/the-prime-excalibur/
```

Or download the source from the repository:

```text
https://github.com/Artifactsmaker/the-prime-excalibur
```

### AI Setup

- Cloud providers such as Gemini require your own API key.
- Local presets such as Ollama/Gemma do not require an API key, but Ollama must be installed separately.
- Settings are stored locally in the browser/app storage.

### Included

- Main Operator Intelligence console.
- Prime Overlap Genesis Chamber visual stage.
- Bundled operator library.
- Companion apps: Trajectory Lab, Element Passport, Prime Ulam Atlas.
- Interface language: EN / VN / JP.

## 日本語

The Prime Excalibur は、ローカルで動作する Operator Intelligence コンソールです。選択した AI アシスタントへ送信する前に、同梱された O.i オペレーター仕様を使ってプロンプトを構造化します。

### 起動方法

Web 版を直接利用できます。

```text
https://artifactsmaker.github.io/the-prime-excalibur/
```

### AI 設定

- Gemini などのクラウド provider には、ユーザー自身の API key が必要です。
- Ollama/Gemma などの local preset は API key 不要ですが、Ollama は別途インストールが必要です。
- 設定はローカル環境に保存されます。

### 同梱内容

- メイン Operator Intelligence コンソール。
- Prime Overlap Genesis Chamber ビジュアルステージ。
- 同梱オペレーターライブラリ。
- 連携アプリ: Trajectory Lab、Element Passport、Prime Ulam Atlas。
- UI 言語: EN / VN / JP。
