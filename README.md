# The Prime Excalibur

**The Prime Excalibur** is a static HTML Operator Intelligence console. It runs in the browser, applies bundled O.i operator specifications, and sends the final prompt to Gemini with the user's own API key.

Direct web version:
https://artifactsmaker.github.io/the-prime-excalibur/

Repository:
https://github.com/Artifactsmaker/the-prime-excalibur

ORCID: 0009-0000-7520-6781

DOI: 10.5281/zenodo.21310036

## Run

Open `index.html` directly, or upload the whole folder to GitHub Pages.

For the public version, use:

```text
https://artifactsmaker.github.io/the-prime-excalibur/
```

## Gemini API Key

The app uses Gemini by default. Open the Setup tab, paste your Gemini API key, then press Apply.

Get a free key:

```text
https://aistudio.google.com/app/apikey
```

The key is stored only in local browser storage.

## Included

- Main Operator Intelligence console.
- Prime Overlap Genesis Chamber visual stage.
- Bundled O.i operator library.
- Companion apps: Trajectory Lab, Element Passport, Prime Ulam Atlas.
- Interface language: EN / VN / JP.
